
****************************************** Instrucciones para desplegar el proyecto **********************************

1) descargar el proyecto desde el repositorio
2) descomprimir el proyecto en el computador local
3) Abrir el Ide de su preferencia
4) ir a file y dar clic en open, posterior a eso buscar la ruta donde se dejo el proyecto
5) Busca el archivo Gradle del proyecto y dar doble clic
6) las excepciones se manejan desde un archivo de excel en la raiz del proyecto, esto con el fin de que sea 
manipulado por analistas funcionales
